# SudokuSolver
Sudoku Solver Java
1.0

This Solver is 1.0, In this program it uses a 2D array.

Format:
                {
                {6, 4, 9, 1, 5, 7, 0, 0, 2},
                {8, 0, 1, 0, 9, 6, 0, 4, 0},
                {3, 0, 5, 0, 8, 4, 1, 0, 0},
                {4, 3, 7, 9, 2, 8, 5, 6, 1},
                {5, 1, 8, 7, 6, 3, 4, 2, 9},
                {2, 9, 6, 4, 1, 5, 3, 0, 0},
                {1, 5, 3, 6, 7, 2, 9, 8, 4},
                {9, 0, 4, 8, 3, 1, 2, 5, 7},
                {7, 0, 2, 5, 4, 9, 6, 1, 3}
                };
               
               
It does 3 checks to see if the Sudoku is solvable.
